# Portfolio — Khadim SEYE

lien : https://portfolio-khadim-navy.vercel.app/


Portfolio professionnel construit avec **React + TypeScript + Vite + Tailwind CSS**.

## 🚀 Démarrage rapide

```bash
npm install
npm run dev
```

## 📦 Build production

```bash
npm run build
npm run preview
```

## 🌐 Déploiement Vercel

1. Pousser le projet sur GitHub
2. Connecter le repo sur [vercel.com](https://vercel.com)
3. Vercel détecte Vite automatiquement → Deploy !

## 📁 Structure

```
src/
├── data/           # Données centralisées (profil, projets, formations, certifs)
├── components/     # Composants réutilisables (RootLayout, CarteAttestation)
├── pages/          # Pages (Home, Projects, Education, Certifications, Contact)
└── main.tsx        # Point d'entrée
public/
└── photo.jpeg      # Ta photo de profil
```

## ✏️ Personnalisation

Toutes tes données sont dans `src/data/` — modifie uniquement ces fichiers pour mettre à jour le portfolio.
